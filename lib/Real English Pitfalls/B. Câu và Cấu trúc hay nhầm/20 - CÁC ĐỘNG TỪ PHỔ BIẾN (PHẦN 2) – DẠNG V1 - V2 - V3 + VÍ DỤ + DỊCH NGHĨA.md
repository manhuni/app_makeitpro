# 🔥 CÁC ĐỘNG TỪ PHỔ BIẾN (PHẦN 2) – DẠNG V1 - V2 - V3 + VÍ DỤ + DỊCH NGHĨA

---

## 21. **Buy** – bought – bought
- ❗ Nghĩa: mua
- 📝 Ví dụ:
  - She bought a new phone. → Cô ấy đã mua một chiếc điện thoại mới.
  - I’ve bought everything we need. → Tôi đã mua mọi thứ ta cần.
